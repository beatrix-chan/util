# Tex Renderers

Allows inputting an equation in $\TeX$ format using [MathJax](https://www.mathjax.org/) or [KaTeX](https://katex.org/)

## Technology

- HTML5 and [Bootstrap5](https://getbootstrap.com/)
- [jsDelivr CDN](https://www.jsdelivr.com/) links for [MathJax](https://www.jsdelivr.com/package/npm/mathjax) and [KaTeX](https://www.jsdelivr.com/package/npm/katex)
